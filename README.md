# Chicago Taxi Data Analysis

## Project Description

This project analyzes taxi trip data in Chicago to understand passenger behavior and the impact of weather conditions on trip duration.

The analysis includes SQL queries, Python exploratory data analysis, data visualization, and statistical hypothesis testing.

## Tools Used

- Python
- Pandas
- Matplotlib
- SciPy
- SQL
- BeautifulSoup
- Jupyter Notebook

## Project Steps

1. Web scraping weather data
2. SQL exploratory analysis
3. Data visualization
4. Hypothesis testing

## Key Findings

- Flash Cab had the highest number of trips.
- The Loop neighborhood had the highest number of trip drop-offs.
- Weather conditions have a statistically significant effect on trip duration.
